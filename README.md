# Agendei

Agendei é uma aplicação mobile desenvolvida com **React Native** que permite aos usuários marcarem consultas médicas de forma simples e rápida. A aplicação foi projetada para facilitar o agendamento de consultas e melhorar a experiência do usuário.

## Funcionalidades

- **Cadastro de Usuários**: Permite que os usuários criem suas contas para acessar a aplicação.
- **Agendamento de Consultas**: Usuários podem marcar consultas médicas de acordo com a disponibilidade dos profissionais.
- **Lembretes de Consultas**: Notificações para lembrar os usuários sobre as consultas agendadas.
- **Histórico de Consultas**: Os usuários podem visualizar suas consultas passadas e futuras.
- **Interface Intuitiva**: Navegação fácil e acessível para todos os usuários.

## Tecnologias Utilizadas

- React Native: Para o desenvolvimento da aplicação mobile.
- Axios: Para fazer requisições HTTP.
- React Navigation: Para navegação entre telas.
- SQLite: Para armazenamento de dados local.

## Instalação

Para instalar e executar a aplicação, siga os passos abaixo:

1. Clone o repositório:
   ```bash
   git clone 
   ```

2. Navegue até o diretório do projeto:
   ```bash
   cd agendei-mobile
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```


5. Abra no emulador ou dispositivo físico com React Native:
   ```bash
   npm expo-start
   ```
